# Quick Start

## 1) Install
```bash
git clone https://github.com/zunaidi85/zstack-php.git
cd zstack-php
cp .env.example .env
```

## 2) Run
Point your web root to the `public/` directory.

Visit `/` to verify installation.

## Demo Login (optional)
1) Configure DB in `.env`
2) Run:
- `database/schema.sql`
- `database/seed.sql`
3) Login:
- `test@example.com` / `secret123`
